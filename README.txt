Преподаватель:

Сложность задания - уровень вложенности
После определения сложности задания, надо нажать на кнопку генерации унарной, либо бинарной функции
Есть окно предпросмотра функции, если Вас устраивает функция, можно добавить её в список заданий. Если функция не устраивает, её можно обновить
Если в какой-то момент, Вы решили, что список задач вас не устраивает, его можно очистить.
После составления списка, его надо зашифровать. Зашифрованную строку нужно скопировать и отправить студенту

Студент:

Нужно ввести зашифрованную строку, отправленную преподавателем и нажать на кнопку "Узнать задания", для отображения списка задач.
